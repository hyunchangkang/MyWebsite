Human-Robot Collaboration System

Hyun Chang Kang, Collaborators
2023-2024

Project Overview

Safety-critical reactive motion control system for collaborative robotics using proximity sensors and adaptive control algorithms. This system enables safe and intuitive human-robot interaction in shared workspaces.

Key Features

  • Real-time proximity detection using capacitive and inductive sensors
  • Variable admittance control for smooth interaction
  • Safety-critical collision avoidance algorithms
  • Adaptive control based on human behavior prediction

Technical Details

The collaboration system employs dual-type proximity sensors to detect human presence and intention, enabling proactive safety measures. The constrained variable admittance control ensures smooth and natural interaction while maintaining strict safety boundaries.

Results

Experimental validation showed significant improvement in safety metrics and user experience compared to traditional force-feedback systems, with reduced collision incidents and enhanced collaborative task performance.
